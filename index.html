Featured Project

Featured Project: Our Lady at the Pillar School

Complete network infrastructure transformation and modernization

Network Infrastructure Improvements

Significant improvements have already been made to the core of our network:

  • Increased backbone bandwidth
  • Redundant uplinks and failover systems to reduce downtime
  • Improved routing and stability
  • Upgrades to the network closets in both the school and the office

This project involved converting a complete network stack to a Ubiquiti network stack, utilizing proper server racks, cleaning up wiring, and organizing the infrastructure for optimal performance and maintainability. Additionally, we migrated their existing IP phone solution to a unified UniFi Talk solution, upgrading to the latest phones while preserving some existing equipment to optimize costs.
